# Anonymous Telegram Bot

[![eng](https://img.shields.io/badge/lang-en-en?color=DCDCDC)](https://github.com/allcodny/anon_tgbot/blob/master/README.md)
[![rus](https://img.shields.io/badge/lang-ru-ru)](https://github.com/allcodny/anon_tgbot/blob/master/README-ru.md)

A simple Telegram bot for feedback

## ✨ Features:
* Forward messages from a user to an administrator
* Forward media groups
* Forward files and other types of messages
* Block a user

## ⚙️ Installation:
1. Download the `bot.py` file
2. Install the `aiomysql` and `aiogram` libraries
3. Change the values ​​in the variables to your own.
4. Done!
